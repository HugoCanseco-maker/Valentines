## Run Locally

```bash
npm install
npm run dev
```

Then open [http://localhost:5173](http://localhost:5173)

## Build for Production

```bash
npm run build
```

The output will be in the `dist/` folder. Deploy to any static hosting (Vercel, Netlify, GitHub Pages, etc.).
